# Drop your brand assets in this folder

Files expected by the website (see /README.md for the full table):

- logo.png              → 240 × 60 px, PNG transparent (navbar logo)
- favicon.ico           → 32 × 32 px (multi-size .ico)
- favicon.png           → 192 × 192 px (PNG transparent)
- apple-touch-icon.png  → 180 × 180 px (PNG, square, no transparency)

If logo.png is missing, the site automatically falls back to a styled red "A" mark.
